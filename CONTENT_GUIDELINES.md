# Blog Content Guidelines

Guidelines for creating and formatting blog posts for the TaskFlow blog.

## Content Structure

### Frontmatter Template

```yaml
---

date: 2025-01-15
image: /images/your-image.png

seoTitle: "SEO Optimized Title | TaskFlow"
seoDescription: "SEO description for meta tags (150-160 characters)"
---
```

## Formatting Rules

### Paragraphs

- **Length**: 2-4 sentences per paragraph
- **Spacing**: Blank line between paragraphs
- **Structure**: One main idea per paragraph

### Headings

- **H2 (`##`)**: Major sections (Introduction, Conclusion, main topics)
- **H3 (`###`)**: Subsections within major sections
- **H4 (`####`)**: Minor subsections (use sparingly)
- **Spacing**: Add horizontal rule (`---`) before major H2 sections

### Lists

- **Bullet lists**: For unordered items
- **Numbered lists**: For sequential steps
- **Spacing**: Proper indentation and spacing between items

### Emphasis

- **Bold (`**text**`)**: For key concepts, important terms
- **Italic (`*text*`)**: For emphasis, quotes
- **Code (`` `text` ``)**: For technical terms, code references

### Images

- **Format**: `![Descriptive Alt Text](/images/filename.png)`
- **Placement**: After major sections or relevant content
- **Alt text**: Always descriptive and meaningful

### Horizontal Rules

Use `---` (three dashes) between major sections for visual separation.

## SEO Guidelines

### Title Optimization

- Include primary keyword
- Keep under 60 characters
- Make it compelling and clickable
- Include brand name when appropriate

### Description Optimization

- 150-160 characters
- Include primary and secondary keywords
- Write compelling copy that encourages clicks
- Include a call to action when appropriate

### Content Optimization

- **Word count**: Aim for 2000-3000 words for comprehensive posts
- **Keyword density**: Use keywords naturally (1-2% density)
- **Headings**: Include keywords in H2 and H3 headings
- **Internal linking**: Link to other relevant blog posts
- **External linking**: Link to authoritative sources

### Tag Strategy

- Use 3-6 relevant tags per post
- Include both broad and specific tags
- Use consistent tag names across posts
- Focus on tags your audience searches for

## Writing Style

### Tone

- **Professional but approachable**: Write like you're talking to a colleague
- **Actionable**: Provide practical, implementable advice
- **Evidence-based**: Support claims with research or examples
- **Clear and concise**: Avoid jargon, explain technical terms

### Structure

1. **Hook**: Compelling opening that addresses reader's problem
2. **Context**: Provide background and set expectations
3. **Content**: Deliver value with clear sections
4. **Examples**: Include real-world examples and use cases
5. **Action**: Provide actionable steps or takeaways
6. **Conclusion**: Summarize and provide next steps

### Best Practices

- **Start with the problem**: Address what readers are struggling with
- **Provide solutions**: Offer actionable advice
- **Use examples**: Make concepts concrete with examples
- **Break it down**: Use lists, headings, and short paragraphs
- **Include visuals**: Add images, diagrams, or screenshots
- **End with value**: Leave readers with actionable takeaways

## Image Guidelines

### Image Specifications

- **Featured images**: 1200x630px (recommended for social sharing)
- **Content images**: Max 1200px width
- **Format**: PNG for graphics, JPG for photos
- **File size**: Optimize to under 500KB when possible

### Image Naming

- Use descriptive, lowercase names with hyphens
- Include relevant keywords when appropriate
- Examples:
  - `deep-work-focus.png`
  - `productivity-tips.png`
  - `task-management-workflow.png`

### Image Placement

- **Featured image**: In frontmatter, displayed at top of post
- **Content images**: After relevant sections or paragraphs
- **Alt text**: Always include descriptive alt text

- [ ] Paragraphs are 2-4 sentences
- [ ] Images have descriptive alt text

- [ ] Tags are relevant and consistent
- [ ] SEO fields are optimized
